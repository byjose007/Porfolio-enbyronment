Angular · Architecture Lectura 5 min 5 min read

Por qué sigo apostando por Angular para proyectos a largo plazo ↗ Why I still bet on Angular for long-term projects ↗

La estructura de Angular puede parecer muy estricta al principio, pero he comprobado cómo esas reglas predefinidas ayudan a los equipos a mantener el orden y escalabilidad en aplicaciones gigantes durante años. Angular's strict structure might seem rigid at first, but I've proven how these rules help teams maintain order and clean code in large-scale applications over the years.

Reactivity · Signals Lectura 5 min 5 min read

Signals y RxJS: usar cada uno donde realmente aporta valor ↗ Signals and RxJS: Using each where they truly add value ↗

RxJS es una herramienta excelente para manejar eventos asíncronos complejos y flujos de datos distribuidos. Pero para reactividad local de componentes, las Signals nos ofrecen un paradigma mucho más directo y limpio. RxJS is an excellent tool for handling complex asynchronous events and stream coordination. But for local component state, Signals offers a much simpler and more direct reactivity model.

AI · Developer Experience Lectura 6 min 6 min read

Cómo integrar copilotos de IA sin perder el control del proyecto ↗ How to integrate AI co-pilots without losing project control ↗

Uso copilotos inteligentes como Claude Code para acelerar tareas repetitivas o prototipado veloz, pero evito delegarles decisiones de arquitectura estructural. La IA acelera el código que entiendes — y puede ocultar lo que no. I use advanced AI tools like Claude Code to speed up repetitive tasks and scaffolding, but avoid delegating architectural control to them. AI accelerates the code you understand — it can conceal what you don't.

UX/UI · Interaction Lectura 5 min 5 min read

La gran diferencia entre una pantalla correcta y una útil ↗ The major difference between a correct screen and a useful one ↗

En pruebas de usabilidad con operadores aéreos descubrimos que la densidad informática no era la meta. Un operario bajo estrés no lee datos celda por celda; requiere escanear visualmente y capturar anomalías rápido. In early testing with aviation operators, we discovered that showing all raw data wasn't the goal. A user under heavy pressure doesn't read every cell; they need to scan visually and detect patterns instantly.

Cognitive Load · Forms Lectura 5 min 5 min read

Reducir la carga cognitiva en formularios empresariales complejos ↗ Reducing cognitive load in complex enterprise forms ↗

Los problemas de interfaz no siempre se solucionan refinando colores o tipografías. En un sistema crítico de riesgos laborales, la clave del éxito fue la revelación progresiva y paulatina de campos según el contexto operativo. Sometimes interface problems are not solved by refining colors or typography. In a highly regulated safety compliance system, the key was progressive disclosure — showing inputs only when contextually relevant.

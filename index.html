03 — Sobre mí

Interfaces limpias para sistemas complejos.

Soy desarrollador frontend especializado en Angular, TypeScript y microfrontends. Mi foco es diseñar e implementar interfaces de alta fidelidad que actúan como escudo frente a la complejidad en dominios críticos (aviación, banca, oncología), garantizando que el usuario tome decisiones acertadas bajo presión.

Para mí, un gran frontend es un sistema predecible. Lo consigo mediante arquitecturas modulares bien estructuradas, accesibilidad WCAG 2.1 AA por diseño, y capas BFF (NestJS/Python) que simplifican el consumo de datos. Además, optimizo el ciclo de desarrollo incorporando IA de forma pragmática, asegurando que cada línea de código sea tan escalable como segura.
